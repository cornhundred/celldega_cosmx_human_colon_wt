# CosMx Human Colon (WTx) — Celldega DegaFiles

Celldega **LandscapeFiles (DegaFiles)** for a CosMx SMI Whole-Transcriptome colon CRC sample (S0),
generated by converting the CosMx flat files into the Xenium format and running the existing
`celldega.pre` Xenium pre-processing (classic, non-row-group layout). All ~1.12B transcripts are
included (streamed, not subsampled).

Contents (repo root):
- `landscape_parameters.json` — technology=Xenium, classic (non-row-group) layout
- `cell_metadata.parquet`, `meta_gene.parquet`, `df_sig.parquet`, `cell_clusters/`
- `cbg/` — one parquet per gene (18,935 genes)
- `transcript_tiles/` — one parquet per spatial tile (all transcripts)
- `cell_segmentation/` — one parquet per spatial tile (cell boundary polygons)
- `pyramid_images/` — DeepZoom (.webp) pyramids for DNA / PanCK / Membrane / CD45
- `micron_to_image_transform.csv`

## View
```python
import celldega as dega
base_url = "https://raw.githubusercontent.com/cornhundred/celldega_cosmx_human_colon_wt/main"
landscape = dega.viz.Landscape(technology="Xenium", base_url=base_url, height=700)
landscape
```

Source data: NanoString CosMx colon CRC discovery release. Morphology images are vertically
flipped from the Napari OME-ZARR to align with the CosMx `global_px` coordinate frame.
